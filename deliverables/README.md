## How the POC maps to Topic 1

| Topic 1 technology | Where it shows up in the POC |
|---|---|
| Computer Vision | DenseNet-121 image encoder (`backend/inference.py`) |
| NLP / text generation | Transformer decoder produces the narrative report |
| LLM / LMM + grounding | RAG clinical agent: ClinicalBERT + ChromaDB + LangChain ReAct (`backend/rag_agent.py`) |
| Deployment / MLOps | FastAPI, Celery, Prometheus, OpenTelemetry, Docker, Gradio UI |
